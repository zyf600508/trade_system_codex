# 资金流向和筹码分布监控交易系统（V1）

## 开发计划

1. **阶段一：可运行骨架（已完成）**
   - 配置中心（统一参数管理）
   - SQLite 数据库与表结构
   - 股票池筛选
   - 模拟数据采集与增量更新
   - 指标评分与买卖建议输出
2. **阶段二：真实数据源接入（进行中）**
   - 主备数据源适配器
   - 字段映射与数据质量校验
3. **阶段三：回测与参数调优（待做）**
   - 回测框架
   - 参数扫描与阈值自动推荐
4. **阶段四：实盘执行与监控（待做）**
   - 券商交易接口
   - 告警推送与可视化看板

## 当前能力

- 初始化近一年的日线、资金流向、筹码指标（模拟数据）。
- 每次运行自动追加最新交易日数据。
- 计算综合评分并输出 Top10 操作建议。
- 风险控制支持止盈（30%）、止损（-15%）与评分卖出提醒。
- 日志同时写入日志文件和数据库。

## 快速开始

```bash
python run.py
```

执行后会生成：
- `trade_system.db`：系统数据库
- `trade_system.log`：中文运行日志

## 目录结构

```text
src/trade_system/
  app.py          # 系统主流程
  config.py       # 配置加载
  data_provider.py# 数据源（当前为模拟）
  database.py     # 数据库初始化
  portfolio.py    # 持仓与风险控制
  repository.py   # 数据写入仓储
  scoring.py      # 评分与动作映射
  logger.py       # 日志
config.json       # 统一配置文件
run.py            # 启动脚本
```
